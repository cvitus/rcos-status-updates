Github stuff:

Download a repository:
git clone URL.git


If you add a new file:
git add FILE_NAME


Commit your changes: (make changes)
git commit -a -m "YOUR MESSAGE OF WHAT YOU CHANGED"


Push your commits: (saves changes everywhere/ upload changes)
git push -u origin master
